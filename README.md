# 🎬 Sentiment Analysis on IMDB Movie Reviews

## 📌 Project Overview
This project performs sentiment analysis on 50,000 IMDB movie reviews,
classifying them as Positive or Negative using Machine Learning.

## 📊 Dataset
- Source: IMDB Dataset from Kaggle
- Size: 50,000 reviews
- Balance: 25,000 Positive | 25,000 Negative

## Tech Stack
- Python
- Pandas and NumPy
- NLTK
- Scikit-learn
- Matplotlib and Seaborn

## Results

| Model               | Accuracy |
|---------------------|----------|
| Logistic Regression | 89%      |
| Naive Bayes         | 86%      |
| Random Forest       | 85%      |

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Open sentiment_analysis_imdb.ipynb in Jupyter
3. Run all cells from top to bottom
